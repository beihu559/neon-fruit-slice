# 霓虹切水果

深色霓虹风格的 HTML + Canvas 切水果小游戏，单文件即可运行。

## 玩法

- 水果从底部抛物线弹起，划动切开得分
- 避开黄黑条纹炸弹（切到扣命，共 3 命）
- **30 秒极速** / **60 秒经典** / **60 秒掉果扣分**（漏切扣 5 分，得分最低 0）
- 结束后可换模式再玩一次
- 触控板轻滑即可切，无需按住

## 运行

直接在浏览器中打开 `index.html`，或使用 GitHub Pages 在线游玩。

## 技术

- 纯 HTML / Canvas / JavaScript
- `requestAnimationFrame` 游戏循环
- 无外部依赖
